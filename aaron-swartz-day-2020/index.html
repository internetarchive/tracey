It volume mounts the dirs into your container upon startup. We'll simulate that _first_ ... and _then_ put tracey out of a job with _10 lines of code_ --- ### setup ```bash mkdir /task /item wget -qO /item/stairs.avi https://archive.org/download/stairs/stairs.avi echo '{"sourceFile": "/item/stairs.avi", "targetFile": "/item/thumb.jpg"}' \ > /task/task.json ``` ### package.json ```json { "type": "module" } ``` ### Dockerfile ```bash FROM node:alpine COPY . /app RUN apk add ffmpeg CMD node /app/derive.js ``` ### derive.js ```javascript import { readFileSync } from 'fs' import { spawnSync } from 'child_process' const task = JSON.parse(readFileSync('/task/task.json').toString()) spawnSync('ffmpeg', ['-i', task.sourceFile, task.targetFile]) ```
